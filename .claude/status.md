# 문제 유형별 구현
- 사지선다: 선택 + 확인 → 정답/오답 + 해설
- 복수선택: 여러 개 선택 + 확인
- OX: O/X 버튼 + 확인
- 순서 맞추기: 클릭으로 순서 번호 매기기 + 확인 (4~5개)
- 2개 순서 선택: ①② 순서로 선택 + 확인
- 선 잇기: 왼쪽→오른쪽 교대 클릭 쌍 맺기 (3~4쌍)
- 십자말풀이: 격자 + 2개 보기 중 선택 → 격자에 글자 채움
